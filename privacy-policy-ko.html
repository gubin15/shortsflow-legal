ShortsFlow 개인정보 보호정책

최종 업데이트: 2024년 11월 18일

1. 소개

ShortsFlow(이하 "본 앱")를 이용해 주셔서 감사합니다. 저희는 귀하의 개인정보를 존중하며 개인정보 보호에 최선을 다하고 있습니다. 본 개인정보 보호정책은 정보의 수집, 사용, 저장 및 보호 방법을 설명합니다.

2. 수집하는 정보

2.1 귀하가 제공하는 정보

  • Google 계정 정보: Google OAuth로 로그인할 때 Google 계정 이메일, 사용자 이름, 프로필 사진을 수집합니다
  • YouTube 채널 정보: 채널 ID, 채널 이름, 구독자 수 등의 공개 정보

2.2 자동으로 수집되는 정보

  • YouTube 동영상 데이터: YouTube Data API를 통해 얻는 동영상 제목, 설명, 조회수, 좋아요 수, 댓글 수, 게시 날짜 등
  • 사용 데이터: 앱 기능 사용 현황, 작업 로그(로컬 저장만)
  • 기기 정보: 기기 모델, OS 버전(앱 성능 최적화용)

2.3 수집하지 않는 정보

  • YouTube 계정 비밀번호는 수집하지 않습니다
  • 결제 정보는 수집하지 않습니다
  • 위치 정보는 수집하지 않습니다
  • 연락처 정보는 수집하지 않습니다

3. 정보 사용 목적

수집된 정보는 다음 목적으로 사용됩니다:

  • 핵심 기능 제공: 동영상 관리, 데이터 분석, 통계 보고서
  • 스마트 인사이트: 동영상 성과 분석, 최적화 제안 제공
  • 사용자 경험 개선: 앱 성능 및 기능 개선
  • AI 기능: Google Gemini AI를 사용한 콘텐츠 제안 생성(선택 사항)

4. 정보 저장

4.1 로컬 저장

  • 모든 동영상 데이터는 귀하의 기기의 로컬 SQLite 데이터베이스에 저장됩니다
  • 로그인 인증 정보(토큰)는 기기의 보안 저장소에 암호화되어 저장됩니다
  • 앱 설정 및 환경설정은 로컬에 저장됩니다

4.2 클라우드 저장

  • 귀하의 데이터를 저희 서버에 업로드하지 않습니다
  • 모든 데이터는 귀하의 기기에만 저장됩니다

5. 제3자 서비스

5.1 Google OAuth

  • 목적: 사용자 인증
  • 개인정보 보호정책: Google 개인정보 보호정책

5.2 YouTube Data API

  • 목적: YouTube 동영상 데이터 가져오기
  • 권한: 읽기 전용, YouTube 콘텐츠 수정 불가
  • 서비스 약관: YouTube 서비스 약관

5.3 Google Gemini AI

  • 목적: 콘텐츠 최적화 제안 생성(선택 사항)
  • 개인정보 보호정책: Google 개인정보 보호정책

6. 데이터 보안

  • 암호화 저장: 민감한 정보는 암호화되어 저장됩니다
  • HTTPS: 모든 네트워크 요청은 HTTPS 암호화를 사용합니다
  • 액세스 제어: 귀하만 데이터에 액세스할 수 있습니다
  • 정기 업데이트: 보안 패치를 신속하게 적용합니다

7. 귀하의 권리

  • 액세스 권한: 언제든지 앱 내의 모든 데이터를 볼 수 있습니다
  • 내보내기 권한: 데이터를 내보낼 수 있습니다(CSV, JSON, Excel 형식)
  • 삭제 권한: 설정에서 모든 로컬 데이터를 지울 수 있습니다
  • 취소 권한: 언제든지 Google 계정 인증을 취소할 수 있습니다

8. 아동 개인정보 보호

본 앱은 13세 미만 아동을 대상으로 하지 않습니다. 13세 미만 아동으로부터 정보를 수집한 것으로 확인되면 즉시 삭제합니다.

9. 개인정보 보호정책 업데이트

본 개인정보 보호정책은 수시로 업데이트될 수 있습니다. 중대한 변경 사항이 있을 경우 앱 내에서 알려드립니다. 계속 사용하면 업데이트된 정책에 동의한 것으로 간주됩니다.
